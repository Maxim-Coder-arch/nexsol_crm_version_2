в папке dashboard будут лежать все странички, будут доступны при авторизации, внутри dashboard будут лежать папки со страничками - одна папка = одна страничка. Внутри одно папки со страничкой будут лежать index/page.tsx и index.module.scss, а также вложенная папка ui, где будет сама верстка, так как в index/page будет лежать логика, а пропсами данные буду перебрасываться в компоненты в папке ui



global-styles - папка с базовыми стилями, там ничего трогать не нужно

dashboard - папка с защищенными (для авторизованных пользователей) страничками. Там и будет все странички

components - папка, в которой хранятся ui компоненты, например меню, модальные окна, кнопки и т.п.
