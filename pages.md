установка статусов заявок, удаление заявок

добавление заявок вручную (если клиент связался через соц сети напрямую. Пригодится для правилного расчета конверсии)

автоматический расчет конверсии на основе статусов заявок

форма для ручного добавления заявок (с ручными заявками точно также, как и с зхаявками с сайта: изначально при добавление стстаус будет новым)

4 страница (clients) - управление базой данных клиентов


фиксирование клиентов и их данных

рабочий статус клиента: новый, в работе, завершенный

физический статус клиента: успешный, потерянный. Все успешные клиенты сортируются по обычным столбикам, а клиента с не успешным статусом идет в отдельную колонку "потерянные"


карточка с клиентом:
- имя клиента
- функции изменения рабочего статуса клиента (изначально новый)
- функции изменения физического статуса клиента (изначально успешный статус)
- дата добавления карточки с клиентом (генерируется автоматически, когда карточка будет добавлена - такая дата и появится)
- комментарий по сделке
- дополнительные данные (ключ - значение)
- кнопка удаления карточки с клиентом
- функция редактирования карточки клиента
